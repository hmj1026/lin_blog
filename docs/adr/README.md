## ADR 格式

每個 ADR 應包含：
1. **標題**：簡短描述決策
2. **狀態**：提議中 / 已採納 / 已廢棄 / 已取代
3. **上下文**：為什麼需要這個決策
4. **決策**：我們決定做什麼
5. **後果**：這個決策的影響
